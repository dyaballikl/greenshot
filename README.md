Fork from Greenshot -- https://github.com/greenshot/greenshot<br>
This fork enabled tabbed screenshot in the editor, similar to the style of Snagit. Seems this feature has been requested in Greenshot for over a decade and never happened.
******************

How to use:<br>
I'll keep it simple. Clone this repo to your computer (google how to clone a github repo if necessary, it's easy).<br>
Within that destination folder, this is the executable folder:<br>
>\src\Greenshot\bin\Release\net480

Couple of options you have:<br>
    1. Move that folder into your program files (x86) folder<br>
    2. Leave it there<br>
Create a shortcut of the greenshot.exe file within that folder.<br>
Hit Windows key + R and type shell:startup and hit OK.<br>
Move the shortcut into that folder to make greenshot automatically open on startup.<br>
You can also just manually run that exe.
