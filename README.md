Greenshot - a free screenshot tool optimized for productivity
=============================================================

******************
Fork from Greenshot -- https://github.com/greenshot/greenshot
This fork enabled tabbed screenshot in the editor, similar to the style of Snagit. Seems this feature has been requested in Greenshot for over a decade and never happened.
******************

How to use:
I'll keep it simple. Clone this repo to your computer (google how to clone a github repo if necessary, it's easy).
Within that destination folder, this is the executable folder:
    \src\Greenshot\bin\Release\net480
Couple of options you have:
    1. Move that folder into your program files (x86) folder
    2. Leave it there
Create a shortcut of the greenshot.exe file within that folder.
Hit Windows key + R and type shell:startup and hit OK.
Move the shortcut into that folder to make greenshot automatically open on startup.
You can also just manually run that exe.
